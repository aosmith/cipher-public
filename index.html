Architecture at a glance

Rails core

Modern Rails 8 stack with Hotwire, Solid Queue, and Propshaft serves the application layer and background jobs.

  • Solid Queue keeps jobs on-box with predictable scheduling.
  • Solid Cache accelerates encrypted timelines without leaking plaintext.
  • Importmap keeps the frontend lightweight and dependency-free.

Peer overlay

Using WebRTC and secure friend invitations, Cipher forms direct mesh links. Metadata is minimized, and relay fallback is optional.

  • Automatic ICE negotiation and TURN opt-in.
  • Session keys rotate per conversation.
  • Offline-first queues flush as peers come online.

Desktop bundling

Tauri wraps the Rails app in a minimal native shell. Rust handles window management while the Rails server runs locally.

  • Single command bin/desktop for development.
  • Platform builds via platform-specific scripts.
  • Custom CSP tuned for the native WebView.

Security posture

Cipher is currently in public beta. Do not store sensitive data until independent audits complete.

  • Cryptography standards follow libsodium and audited Rust crates.
  • Keys never leave the client; backups are opt-in and encrypted.
  • Security guides cover network isolation and hardening tips in /docs.
