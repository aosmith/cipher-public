Architecture at a glance

Rust core

Modern Rust/Tauri stack with SQLite, Ed25519 cryptography, and WebRTC serves the application layer and P2P networking.

  • SQLite keeps data local with comprehensive test coverage.
  • Ed25519/X25519 encryption secures all communications.
  • Pure Rust implementation ensures memory safety and performance.

Peer overlay

Using WebRTC and secure friend invitations, Cipher forms direct mesh links. Metadata is minimized, and relay fallback is optional.

  • Automatic ICE negotiation and TURN opt-in.
  • Session keys rotate per conversation.
  • Offline-first queues flush as peers come online.

Desktop bundling

Tauri wraps the Rust app in a minimal native shell. Cross-platform builds support macOS, Windows, Linux, iOS, and Android.

  • Single command cargo tauri dev for development.
  • Platform builds via cargo tauri build commands.
  • Native performance with WebView frontend.

Build once, deliver everywhere

Web

Run cargo tauri dev to launch the Rust development server with hot-reload and native WebView rendering.

Desktop

cargo tauri build packages the app via Tauri, giving macOS, Windows, and Linux users a native experience.

Mobile

Cross-platform Rust builds support iOS and Android with native performance and the same codebase as desktop.

Security posture

Cipher is currently in private beta. Do not store sensitive data until independent audits complete.

  • Cryptography standards follow Ed25519/X25519 and audited Rust crates.
  • Keys never leave the client; backups are opt-in and encrypted.
  • Comprehensive test suite covers cryptography and P2P security.
